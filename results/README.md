# Curated Results Directory

This directory contains the curated result artifacts intentionally included in
the GitHub repository snapshot.

Included device folders:

- `i7_1165G7/`
- `i7_13700F/`
- `raspberry_pi5/`
- `jetson_orin/onnx/`
- `jetson_orin/trt/`

These artifacts are included because they are part of the research record and
support device-level comparison, paper drafting, and reproducibility checks.

Additional local reruns may create more output files, but those should be
treated as transient unless they are intentionally curated back into this tree.
